### HOW-TO:
1) Create IntelliJ IDEA project:
  - Project: Maven
  - Language: Java
  - Packaging: Jar
  - Java: 17
2) Dependenices:
  - Spring Boot DevTools
  - Spring Data JPA
  - MySQL Driver
  - Spring Web
3) application.properties changes:
  - `spring.datasource.url=jdbc:mysql://localhost:3306/gfgmicroservicesdemo`
  - `spring.datasource.username=put your username here`
  - `spring.datasource.password=put your password here****`
4) Create Entity/Model class
5) Create Repository class
6) Create EmployeeResponse class
7) Create Service class
8) Create Controller class
9) Create Configuration class
10) Create Main class
